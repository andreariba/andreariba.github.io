Codon TASEP

PNAS · 2019

Published

Simulation of ribosome dynamics along mRNA transcripts using an inhomogeneous Totally Asymmetric Simple Exclusion Process (TASEP) with codon-specific elongation rates. Combined with ribosome footprinting data from yeast, the model revealed that amino acid composition is as important a determinant of translation elongation speed as codon–tRNA adaptation — challenging a long-standing assumption in molecular biology.

05

MoonBirths

Open data analysis · 2026

Do births cluster around phases of the Moon? This project extracts every human birth date recorded in Wikidata — over 2.9 million day-precision records spanning two centuries — maps each one to its position in the lunar cycle using astronomical ephemerides, and applies FFT analysis to search for any periodic signal. The result is a clean null finding: moon phase is not a measurable predictor of birth timing.
